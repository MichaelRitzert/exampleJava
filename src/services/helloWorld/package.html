helloWorld is the simplest possible illustration of the client and server sides of an EPICS V4 "service" (based on pvAccess RPC).

The service is the classical "hello world" implemented as a client executable, which a user runs giving their name, and a server which receives the user's name as an argument, and says hello. For instance, in one widow;

% ./helloServerRunner
...

Then in another process...

% ./helloClientRunner greg
        Hello greg
